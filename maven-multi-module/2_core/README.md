BigData
==========

This repository contains some examples for Hadoop and Spark

####Prerequisites:
 
- Download Hadoop 2.7.1
- `export HADOOP_HOME=/opt/soft/hadoop`
- `export PATH=...:$HADOOP_HOME/bin:$HADOOP_HOME/sbin`
- `export HADOOP_CONF_DIR=$PROJECT_PATH/hadoop_conf`
- `hdfs getconf -namenodes`  (checks hadoop configuration)
- `hadoop namenode -format`
- `start-dfs.sh`
