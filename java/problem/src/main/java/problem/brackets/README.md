#### Problem
For string like `()()(()))))))` consisting from `(` and `)`
find a min k, where number of `(` before k equals number of `)` after k 
```
'(((((       ))))))'
 opening k=5 closing 
```

#### Examples
```
"(()).))(" k=4
""         k=0
"(.)"      k=1
"((.))"    k=2
")(.)"     k=2
")(.)("    k=2
"))(.)("   k=3
```
